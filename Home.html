Overview

Chorkor Mobile Doctor is a smart Android app for fitness, AI-powered diagnosis, pharmacy ordering, and real-time drug delivery. It’s built to help people in communities like Chorkor easily access modern healthcare tools from their phone.

Core Features

1. Login & Signup

Supports Apple, Google, and Email/Password login with optional medical history setup.

2. Health Tracker

Counts steps, tracks calories, and displays progress using professional charts like Samsung Health.

3. AI Diagnosis

Users input symptoms and receive AI health analysis with urgency advice and suggested treatment.

4. Pharmacy + Delivery

Users can buy medications, upload prescriptions, and track delivery using Google Maps and Bolt API.

Technology Stack

  • Frontend: Kotlin, Jetpack Compose
  • Backend: Firebase / Node.js + MongoDB
  • Authentication: Firebase Auth
  • AI: OpenAI API / Custom ML
  • Maps: Google Maps API
  • Delivery: Bolt Ride API

Development Phases

  1. UI & Login Design – 2–3 weeks
  2. Step Tracker Integration – 2 weeks
  3. AI Diagnosis Tool – 2 weeks
  4. Mobile Pharmacy + Orders – 3 weeks
  5. Delivery Tracking System – 2 weeks
  6. Testing & Launch – 2–3 weeks

Monetization

  • Subscription for AI features
  • Delivery & pharmacy commissions
  • Health-related ads (optional)
